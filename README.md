# How Websites Work

This project is a small documentation website built with **Jekyll** and published using **GitHub Pages**.

The website explains the basic concepts behind modern web development and how websites are created and delivered on the internet.

## Live Website

You can view the website here:

https://asmirhrb4t.github.io/

## Topics Covered

- Web development basics (HTML, CSS, JavaScript)
- Static site generators
- Content management systems (CMS)
- Installing WordPress on a Linux server
- How browsers communicate with web servers

## Technologies Used

- Jekyll
- GitHub Pages
- Markdown

## Project Structure

```
_config.yml
index.md
web-basics.md
how-the-internet-works.md
example-code.md
images/
```

## Deployment

The website is automatically deployed using **GitHub Pages** whenever changes are pushed to the repository.

## Purpose

This project was created as part of a school assignment to demonstrate how static websites can be generated and deployed using Jekyll and GitHub Pages.

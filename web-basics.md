# Web Development Basics

Modern websites are built using several core technologies.

## HTML

HTML (HyperText Markup Language) defines the structure of a webpage.  
It describes elements like headings, paragraphs, images, and links.

## CSS

CSS (Cascading Style Sheets) controls the visual appearance of a website.  
It defines colors, layout, spacing, and responsive design.

## JavaScript

JavaScript adds interactivity to websites.  
It allows dynamic content, animations, and communication with servers.

# Ways to Build Websites

There are different approaches to creating websites.

## Static Websites

Static websites consist of pre-generated HTML files that are delivered directly to the browser.

They are usually:

- fast
- secure
- simple to host
